# Jovie

A Next.js 14 application with Clerk authentication and Supabase database integration for music artists.

## Features

- Artist profile management
- Social link management
- Music release tracking
- Analytics dashboard
- Feature flag system

## Getting Started

### Prerequisites

- Node.js 20.17 or later
- npm or yarn
- Supabase account
- Clerk account

### Installation

1. Clone the repository
2. Install dependencies:

```bash
npm install
# or
yarn install
```

3. Set up environment variables:

```bash
cp .env.example .env.local
```

4. Start the development server:

```bash
npm run dev
# or
yarn dev
```
